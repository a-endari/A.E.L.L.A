# Installation on macOS

## Prerequisites

- macOS 10.15 (Catalina) or later.

## Installation Steps

1. **Download**: Get the latest `.dmg` file from the [Releases Page](../../releases).
2. **Mount**: Double-click the `.dmg` file to open it.
3. **Install**: Drag the **Universal Language** app icon into the **Applications** folder.
4. **Open**: Go to Applications and double-click **Universal Language**.

> [!NOTE]
> Since this app is not signed by Apple Developer ID yet, you may see a warning.
> To bypass: Right-click the app > Open > Click "Open" in the dialog.

## Troubleshooting

- **Backend not starting**: Ensure you are not running another instance of the app or using port 8000.
